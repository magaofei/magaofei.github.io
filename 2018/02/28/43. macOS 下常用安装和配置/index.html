Homebrew

Homebrew是一个包管理器，就相当于 yum 和 apt ，通过 Homebrew 我们可以在macOS上安装软件。它有几个优点，最重要的就是不需要超级管理员权限，比如默认的 pip 安装一些库的时候，每次都需要 sudo
